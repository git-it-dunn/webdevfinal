1. Group_members.html is located in the html file.
2. Open up the project directory in ubuntu or whatever environment you use to run programs.
3. The libraries express and nodemon are required to run the project. If not installed, install using the npm i express and npm i nodemon commands respectively.
4. Run the nodemon command in your environment.
5. Once the command runs and you see the message Listening for requests on port 3000, go to your default web browser.
6. Type in the url box: localhost:3000.
7. Done, project should now be running on your browser.
8. Group members can also be found in the About Us tab in the navigation bar on top of the page.
